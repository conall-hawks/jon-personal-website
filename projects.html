Cenari.us

My primary project. I have recently dubbed it my Cenari Management System (CMS). As the acronym implies, it's generally for Content Management. However, Cenari.us is many things:

  • Programming snippets, guides, and bookmarks.
  • Security tutorials and blog.
  • Forum
  • Imageboard
  • Social network
  • Personal blog
  • Video game guides and blog.

Its primary purpose is to entertain visitors and monetize my knowledge via Google's Adsense. Its secondary purpose is as a testing ground for experiments in a production environment. Its tertiary purpose is for my own use as a reference on the things I've already learned.

Partyline

Partyline--as the name implies--is a locality-based chatroom. By default, users are dropped into a chatroom that is relative to their geolocation. However, users may create their own rooms or send private messages.

  • Geolocated chatrooms.
  • Private messages.
  • Login system with customizable user profiles.
  • Temporary "anonymous" user profiles.
  • Media uploading.
  • File uploading.
  • Much, much more!

Initially started as a project to learn Meteor. For a while there I think some people were actually using it, but because of a small user base it never caught on. I use it now to quickly establish communication or transfer files to my friends. http://jonhawks.net (Until I can afford to park it at a properly named domain.)
