Filenames in This Directory to Lowercase

REM Whatever directory you run this in will have all its files names converted to lowercase. for /F %%a in ('dir /L /B') do rename %%a %%a

Start Application Minimized

REM Starts an application minimized. START /min "" "iexplore" google.com

Start Application Hidden

' Starts an application hidden. If you wish to kill the task, you must do so through the task manager. ' Note that this is VBScript. Not all applications allowed to be run hidden. Dim objShell Set objShell = WScript.CreateObject ("WScript.shell") objShell.run "mspaint", 0 Set objShell = Nothing
